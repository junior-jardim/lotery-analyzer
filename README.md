# 🎲 LotteryLab

LotteryLab é uma plataforma modular para análise estatística de loterias.

O objetivo do projeto é fornecer ferramentas para análise, visualização e comparação de resultados de diferentes modalidades de loteria, como:

- Mega-Sena
- Lotofácil
- Quina
- Lotomania
- Timemania
- +Milionária

O projeto está sendo desenvolvido em Python com foco em boas práticas de engenharia de software, arquitetura limpa, testes automatizados e interface desktop.
